# blackfire
Blackfire io puppet module.

Installs the blackfire.io php profiler

## Usage

```puppet
class { 'blackfire':
    server_id    => 'gh3214a9-d0la-43qb-8ba3-3va1618710e',
    server_token => '9skm2ixjs51575ba463e813156ed97c85d8ca5c5691db37bbfe36a622a4f'
}
```
